Proyecto Vita

Universidad Politécnica de Juventino Rosas

Presentado por: Cervantes Rodríguez Andrea, Alberto Ramos Ana Paola, Ruiz Coyote María Guadalupe

Asesor: Dr. Juan Israel Yañez Vargas

Fecha: 5 de diciembre de 2024

Resumen

El Proyecto Vita tiene como propósito revolucionar el acceso a productos de higiene femenina en entornos escolares mediante la implementación de un sistema de control automático para dispensar toallas sanitarias gratuitas. Surgió a partir de las dificultades que las mujeres enfrentan para acceder a estos productos en momentos de emergencia, además de la incomodidad de solicitarlos debido a los tabúes sobre la menstruación.

El sistema propuesto utiliza la tecnología RFID en las credenciales estudiantiles como mecanismo de acceso, permitiendo un control preciso sobre el uso individual y ayudando a gestionar el inventario de manera eficiente para reducir el desperdicio de recursos.

Planteamiento del Problema

En México, muchos espacios públicos como universidades y oficinas cuentan con productos básicos como papel higiénico y jabón de fácil acceso. Sin embargo, el acceso a toallas sanitarias es limitado o no está garantizado de manera constante. Esto genera un desafío significativo para las mujeres, ya que a menudo enfrentan situaciones incómodas debido a la dificultad para obtener estos productos en momentos de emergencia.

Adicionalmente, muchas instalaciones no cuentan con un método eficiente para proveer toallas sanitarias de manera fácil, automática y controlada. Los sistemas tradicionales suelen presentar problemas de gestión de inventario, lo que puede provocar desabastecimiento o un uso indebido de los productos.

Objetivos

Objetivo General

Desarrollar e implementar un sistema de dispensación gratuita de toallas sanitarias en universidades y empresas, utilizando tecnología RFID vinculada a credenciales para controlar el uso individual.

Objetivos Particulares

  • Facilitar el acceso a toallas sanitarias en baños de universidades.
  • Reducir el desperdicio mediante un sistema controlado.
  • Fomentar la conciencia sobre la higiene y salud femenina.
  • Diseñar y aplicar encuestas para analizar la viabilidad del sistema.
  • Construir un prototipo funcional del dispensador.

Metodología

El sistema propuesto se basa en la tecnología RFID para identificar a las usuarias y garantizar la distribución eficiente de toallas sanitarias. Se realizaron encuestas para entender la necesidad del sistema y diseñar un dispensador adecuado.

Se empleó un microcontrolador ESP32 para controlar el dispensador, sensores infrarrojos para detectar la disponibilidad de toallas y un sistema de decremento automático del inventario. Se realizaron simulaciones y pruebas con un circuito electrónico funcional.

Resultados

Las encuestas demostraron una alta necesidad de acceso gratuito a toallas sanitarias en espacios educativos. Las pruebas del sistema RFID mostraron un funcionamiento óptimo para controlar la dispensación de los productos.

Los sensores de detección permitieron un monitoreo preciso del inventario, evitando desperdicios y asegurando que las usuarias pudieran acceder a las toallas sanitarias cuando las necesitaran.

Conclusión

El Proyecto Vita representa un paso importante hacia la normalización de la menstruación en espacios educativos, ofreciendo soluciones tecnológicas innovadoras para garantizar la accesibilidad y dignidad de las mujeres.

La implementación del dispensador automático con RFID permite un acceso rápido y sin complicaciones, promoviendo la equidad en el acceso a productos de higiene femenina.
